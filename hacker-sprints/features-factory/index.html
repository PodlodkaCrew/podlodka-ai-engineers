The Foundry

Команда: Alexander Zhurbin, Mikhail Derbichev, Михаил Иконников

Этот проект победил, потому что команда удержала хороший баланс между тем, чтобы реализовать дополнительные фичи сверх базового минимума, и тем, чтобы довести каждую из них до продакшн состояния. Оркестратор выглядит так, что его можно с минимальными усилиями забрать под любой проект.

  • Очень понятный и чистый UI — оркестрацией задач легко управлять, благодаря хорошо проработанному observability на каждом шаге.
  • Хорошо сделан двухуровневый quality gate: сначала детерминистическая проверка линтерами, и только потом запускается LLM-ревьюер.
  • Реализована обработка фидбэка на PR — в той же ветке прогоняется цикл review→fix→push.
  • Работает с разными агентами: Claude, Codex, OpenCode.
  • Реализована долгосрочная память, которая шарится между сессиями и задачами.
  • Перед каждой попыткой реализации делается binary diff чекпойнт — можно восстановиться, если что-то пошло не так.
